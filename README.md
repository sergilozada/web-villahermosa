# Página Web Villa Hermosa

Sitio web informativo desarrollado para presentar el proyecto inmobiliario Condominio Villa Hermosa, mostrando sus beneficios, ubicación, planos, imágenes del proyecto, testimonios y canales de contacto para clientes interesados.

## Descripción

Página Web Villa Hermosa es una aplicación web orientada a la promoción digital del proyecto inmobiliario Condominio Villa Hermosa. Permite mostrar información principal del proyecto, sus beneficios, ubicación, planos generales, imágenes referenciales, testimonios y medios de contacto, facilitando que los visitantes puedan conocer la propuesta y comunicarse con el equipo comercial.

## Funcionalidades principales

- Presentación principal del proyecto Condominio Villa Hermosa
- Información sobre los beneficios del proyecto
- Sección informativa sobre el proyecto
- Visualización de imágenes del condominio
- Galería de imágenes familiares y promocionales
- Sección de ubicación y referencias de acceso
- Visualización de planos generales
- Preguntas frecuentes para clientes interesados
- Testimonios de clientes
- Botón de contacto directo por WhatsApp
- Sección de contacto con asesores comerciales
- Diseño adaptable para computadoras, tablets y celulares

## Tecnologías utilizadas

- React
- TypeScript
- Vite
- CSS
- HTML
- JavaScript
- ESLint

## Objetivo del proyecto

El objetivo de esta página web es fortalecer la presencia digital de Condominio Villa Hermosa, presentando de manera clara, visual y profesional la información comercial del proyecto, sus beneficios, ubicación, planos y medios de contacto, para facilitar que los clientes interesados puedan conocer la propuesta y comunicarse rápidamente con el equipo de ventas.
